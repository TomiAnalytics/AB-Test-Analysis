# A/B Test Analysis: Experiment Reporting

## Objective
Analyze an A/B test to determine whether a website change (button color, layout, etc.) improves user sign-ups.

## Tools & Technologies
Python (Pandas, SciPy, Matplotlib), Excel

## Methodology
1. Data cleaning and preparation  
2. Splitting users into control and test groups  
3. Statistical significance testing (t-test / chi-square)  

4. Visualizing conversion rates  
5. Drawing conclusions from results

## Key Insights
- Conversion rates between control and test groups  
- Statistical significance of observed differences

## Recommendations
- Implement the design if statistically significant  
- Consider further tests for other elements
